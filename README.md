# Sistemi Operativi — UniPD Informatica 2025/2026

Riassunti personali redatti a seguito della frequentazione del corso di **Sistemi Operativi** del corso di laurea in Informatica presso l'Università degli Studi di Padova, anno accademico 2025/2026.

## Contenuto

| Cartella | Descrizione |
|----------|-------------|
| `Riassunti/` | Riassunti delle lezioni, redatti utilizzando tutti i materiali forniti dal docente |
| `Utils/` | Raccolta di funzioni e pattern ricorrenti, ricavati dall'analisi di esami degli anni passati |
| `assets/` | Immagini e diagrammi SVG utilizzati nei file Markdown |

## Visualizzazione

Tutti i file `.md` sono scritti e gestiti tramite **[Obsidian](https://obsidian.md)**. La repository include la configurazione `.obsidian/` per garantire la stessa esperienza visiva a chi utilizza lo stesso editor.

Non posso assicurare la medesima fruizione a chi utilizza visualizzatori diversi (GitHub, VS Code, ecc.): alcune formattazioni o collegamenti interni potrebbero non essere renderizzati correttamente.

## Materiale non incluso

Le slide e il materiale delle lezioni **non sono stati caricati** in questa repository in quanto contrassegnati come materiale UniPD e soggetti a copyright. Sono inoltre esclusi dalla repository i file `Riscritti/`, `Esercizi/`, `Esami/` e i sorgenti Excalidraw in `assets/`.

Chi volesse aggiungere le lezioni localmente può farlo seguendo questi passi:

1. Creare una cartella `Lezioni/` nella root della repository
2. Rimuovere (o commentare) la riga `Lezioni/` dal file `.gitignore`
3. Aggiungere il materiale desiderato e fare push sul proprio account

> **Sconsiglio di lasciare le lezioni come materiale pubblico accessibile a chiunque**, sia per rispetto del copyright UniPD sia per tutelare il lavoro del docente.
